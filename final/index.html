Description

Our project is an interactive web-based fishing game where players catch procedurally generated fish using p5.js. Each fish will have a unique appearance and a quirky, auto-generated name and description created with Tracery. The game features two procedurally generated backgrounds: a fishing dock and an aquarium where players can view their catches. Caught fish are saved to local storage, and players can interact by clicking to catch fish and hovering to view them. Our main focus is on creating a diverse and visually interesting fish generator with a cohesive pixel aesthetic.
